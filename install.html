Installation Instructions for QRAP

To install QRAP, R version 4.0 or greater is required. We also recommend installing R Studio.


Install QRAP From GitHub

QRAP is available on GitHub for all platforms. To install, run:

					## install.packages("devtools") ## you may need install devtools first
					devtools::install_github("goushixue/QRseq")
