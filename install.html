Installation Instructions for QRAP

To install QRAP, R version 4.0 or greater is required. We also recommend installing R Studio.


Install the release version of QRAP

QRAP is available on GitHub for all platforms. To install, run:

					# Enter commands in R (or R studio, if installed)
					# Install the remotes package
					install.packages('devtools')
					devtools::install_github("gsx-ucas/QRAP")

Install the development version of QRAP

Install the development version of QRAP - directly from GitHub.

					# Enter commands in R (or R studio, if installed)
					# Install the remotes package
					install.packages('devtools')
					devtools::install_github(repo = "gsx-ucas/QRAP", ref = "dev")
					library(QRAP)

Get the docker image of QRAP

We provide docker images for QRAP via dockerhub. To pull the latest image from the command line:

					docker pull goushixue/qrap:latest
